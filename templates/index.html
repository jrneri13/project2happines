How did we gather our data?

Now that you all know what inspired our project, we want to show you how we have implemented it:

Data Source: A CSV File, The World Happiness 2017, which ranks 155 countries by their happiness levels. It was released at the United Nations at an event celebrating International Day of Happiness on March 20th, 2018

Data Engineering & Web Visualization: SQLite Database with Python SQL Alchemy to store and retrieve data, Pandas - Dataframes & NumPy for data cleanup and calcs, Flask Routes for JSON, JavaScript to read JSON and display charts, D3.JS, Matplot.JS and Plotly visualizations, along with HTML, CSS and Bootstrap for our content!
